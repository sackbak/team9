VS Code + Git + GitHub Copilot

개발 환경 설정 가이드 - 설치부터 팀 프로젝트 참여까지

PART 1 - 프로그램 설치
1

📦 VS Code 설치

VS Code(Visual Studio Code)는 마이크로소프트에서 만든 무료 코드 에디터입니다.

VS Code 다운로드
  1. 위 링크를 클릭하여 다운로드 페이지로 이동합니다.
  2. "Download for Windows" 버튼을 클릭합니다.
  3. 다운로드된 설치 파일(VSCodeUserSetup-x64-*.exe)을 실행합니다.
  4. 설치 과정에서 기본 설정 그대로 "다음"을 눌러 설치를 완료합니다.
Tip: 설치 시 "PATH에 추가" 옵션이 체크되어 있는지 확인하세요. 터미널에서 code 명령어로 VS Code를 열 수 있게 됩니다.
2

🔧 Git 설치

Git은 소스 코드 버전 관리 도구입니다. 팀원들과 코드를 공유하려면 반드시 필요합니다.

Git 다운로드
  1. 위 링크를 클릭하여 다운로드 페이지로 이동합니다.
  2. "Download for Windows" 버튼을 클릭합니다.
  3. 다운로드된 설치 파일(Git-*-64-bit.exe)을 실행합니다.
  4. 설치 과정에서 기본 설정 그대로 "Next"를 눌러 설치를 완료합니다.
주의: Git 설치 후 VS Code를 한 번 종료했다가 다시 실행해야 Git이 정상적으로 인식됩니다.
PART 2 - Git 초기 설정 (필수!)
3

⚙️ Git 사용자 정보 설정

Git을 처음 사용할 때 반드시 이름과 이메일을 설정해야 합니다. 이 정보가 없으면 git push를 할 수 없습니다.

VS Code에서 터미널 여는 법:

  1. VS Code 상단 메뉴에서 Terminal → New Terminal 클릭
  2. 또는 단축키 Ctrl + ` (백틱) 사용

열린 터미널에 아래 명령어를 한 줄씩 입력하고 Enter를 누르세요:

# 이름 설정 (GitHub 사용자명 권장)
git config --global user.name "여기에본인이름"

# 이메일 설정 (GitHub 가입 이메일과 동일하게)
git config --global user.email "여기에본인이메일@example.com"
중요: 큰따옴표(" ") 안의 내용을 본인 정보로 바꿔주세요!
예시: git config --global user.name "HongGilDong"
예시: git config --global user.email "hong@gmail.com"

설정이 잘 됐는지 확인하려면:

git config --global user.name
git config --global user.email

입력한 이름과 이메일이 출력되면 성공입니다.

PART 3 - 팀 프로젝트 가져오기
4

📂 레포지토리 클론 (Clone)

GitHub에 있는 팀 프로젝트를 내 컴퓨터로 복사(clone)합니다.

  1. VS Code를 실행합니다.
  2. 왼쪽 사이드바에서 "Clone Repository" 버튼을 클릭합니다.
  3. 상단에 나타나는 입력창에 아래 주소를 붙여넣습니다:
https://github.com/AIBizTeam9/team9.git
  1. "Clone from URL" 항목을 클릭합니다.
  2. 폴더 선택 창이 뜨면, 프로젝트를 저장할 로컬 폴더 위치를 지정합니다.
  3. "Select as Repository Destination" 버튼을 클릭합니다.
  4. 클론이 완료되면 "Open"을 눌러 프로젝트를 엽니다.
성공! 왼쪽 Explorer에 프로젝트 파일들 (docs 폴더 등)이 보이면 정상적으로 완료된 것입니다.
PART 4 - GitHub Copilot 설정
5

🤖 GitHub Copilot 활성화

GitHub Copilot은 AI 기반 코드 자동 완성 도구입니다.

  1. VS Code 좌측 하단의 사람 아이콘을 클릭합니다.
  2. "Sign in to GitHub"을 선택하여 GitHub 계정으로 로그인합니다.
  3. 로그인 완료 후, VS Code의 Welcome 탭에서 "Use AI Features with Copilot for free" 항목을 선택합니다.
  4. 우측 하단의 "Use AI Features" 버튼을 클릭합니다.
Copilot Pro ($10/월) 안내:
- 처음 가입 시 30일 무료 체험이 가능합니다.
- 무료 체험 없이도 기본 Copilot Free 기능은 사용할 수 있습니다.
- Pro 요금제는 빡세게 사용하면 3일 만에 토큰을 다 소진할 수 있습니다.
- 토큰은 매월 1일에 리셋됩니다.
PART 5 - 기본 Git 사용법 (참고)
6

🚀 코드 수정 후 Push 하기

파일을 수정한 뒤 GitHub에 올리는 기본 흐름입니다.

# 1. 변경된 파일 확인
git status

# 2. 변경된 파일을 스테이징 (올릴 준비)
git add .

# 3. 커밋 (변경 내용 저장)
git commit -m "변경 내용 설명"

# 4. GitHub에 업로드
git push
Tip: VS Code 왼쪽의 Source Control 탭 (분기 아이콘)에서 GUI로도 동일한 작업을 할 수 있습니다.
